CloudTask - Cloud-Native Task Management Platform

A modern, scalable task management application built with clean architecture principles.

Features
-  User authentication & authorization
-  Task CRUD operations
-  File attachments (AWS S3)
-  Email notifications (AWS SES)
-  RESTful API
-  Responsive React UI
-  Docker support

Tech Stack

**Backend:
- Node.js & Express
- PostgreSQL
- JWT Authentication
- AWS SDK (S3, SES)

**Frontend:
- React 18
- TypeScript
- Tailwind CSS
- Axios

**DevOps:
- Docker & Docker Compose
- GitHub Actions
- Jest Testing

Architecture

This project follows Clean Architecture principles:
- **Entities: Core business models
- **Use Cases: Business logic layer
- **Controllers: Request handlers
- **Routes: API endpoints
